# Contribuindo

Este repositório é um laboratório didático para Subscription Vending no Azure.

Contribuições úteis:

- Correções de documentação.
- Melhorias de validação do request file.
- Ajustes para compatibilidade com novas versões do módulo AVM.
- Exemplos adicionais de product lines.
- Integrações com portal/self-service para o segundo laboratório.

Antes de abrir um pull request:

1. Não inclua secrets, client secrets, arquivos de state ou dados pessoais.
2. Execute `terraform fmt -check -recursive`.
3. Execute `terraform validate` quando possível.
4. Explique no pull request qual problema a alteração resolve.
